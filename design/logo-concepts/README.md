# R.A.W. logo concepts

The selected direction combines the outlined **Pure wordmark** with the
**Practical Systems • Responsible Automation** line from the Workshop block.
It is implemented in the website header mark, favicon and social-sharing image.

1. **Pure wordmark** — restrained, typographic and administrative.
2. **Workshop block** — sturdy, practical and highly legible on vehicles,
   workwear and small icons.
3. **Precision monogram** — technical and structured without using circuit,
   robot or landscape imagery.

Each concept keeps the complete `Randall Automation Works` name with the
`R.A.W.` shorthand to reduce ambiguity with unrelated businesses.
